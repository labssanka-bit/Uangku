LANGKAH 1 — Set RESEND API KEY (WAJIB, 1 menit)
Email dikirim lewat Resend (domain finplansanka.com sudah kamu pakai utk email
Supabase). API key = string diawali "re_".
  • Ambil: resend.com → login → API Keys → (pakai yg ada / Create) → salin.
    (Ini SAMA dgn "password SMTP" yg kamu isi di Supabase Auth → SMTP.)
  • Kirim key itu ke saya, saya set-kan. ATAU jalankan sendiri:

  Di folder project, terminal:
    npx supabase secrets set "RESEND_API_KEY=re_xxxxxxx" --project-ref hsenddeuzmblgbsfwtdp

  Catatan pengirim: email keluar dari noreply@finplansanka.com.
  Domain finplansanka.com harus "Verified" di Resend (biasanya sudah, krn
  dipakai email login). Kalau belum → verifikasi dulu di Resend → Domains.

────────────────────────────────────────
LANGKAH 2 — Pasang webhook di Scalev
Cari menu: Pengaturan / Integrasi / Otomasi / Webhook / Notifikasi.
  • Tambah Webhook baru → tempel URL di atas → event "Order Paid/Lunas".
  • Simpan. Kalau ada tombol "Test/Kirim contoh" → klik (lihat Langkah 3).

  ❓ Kalau Scalev TIDAK punya menu Webhook:
     Cek apakah bisa lewat Zapier/Make/Pabbly (trigger "order paid" →
     action "Webhook POST" ke URL di atas). Kalau tetap tak ada → pakai
     CADANGAN di bawah (auto welcome, kode manual).
