SETUP PRODUK FINPLAN SANKA DI SCALEV + 30 KODE AKSES
============================================================

A. NARUH PRODUK DI SCALEV (bukan cuma link)
Link landing-page-baru-8 itu = halaman LANDING/CHECKOUT. Tetap perlu produk + setting:

1. Buat Produk
   Scalev → Produk → Tambah Produk
   - Nama   : Finplan Sanka — Aplikasi Catatan Keuangan
   - Harga  : 49.900
   - Tipe   : Digital / Non-fisik (tanpa ongkir/kurir)
   - Foto   : pakai logo / 1 kreatif
   - Stok   : 30 (atau sesuai jumlah kode) — biar habis = order ditutup

2. Hubungkan ke halaman checkout
   - Pastikan landing-page-baru-8 terhubung ke produk ini + ada tombol "Pesan/Beli"
     yang mengarah ke form checkout (nama, no HP, email, pembayaran).

3. Pembayaran (WAJIB biar uang masuk otomatis)
   Scalev → Pembayaran → aktifkan: Transfer Bank / QRIS / Payment Gateway.
   Digital → pilih BAYAR DI MUKA (bukan COD). COD tidak cocok utk produk digital.

4. Fulfillment = DIGITAL (kirim kode, bukan kurir)
   Set pengiriman = digital / tanpa pengiriman fisik.

5. Pesan setelah bayar (delivery) → lihat bagian C.

------------------------------------------------------------
B. MEMASUKKAN 30 KODE UNIK

Cek dulu di Scalev apakah ADA fitur "produk digital dengan stok kode/lisensi/voucher unik":
  Produk → (Digital / Lisensi / Kode Unik / Voucher) → tempel daftar kode (1 per baris).
  Kalau ADA → tiap order otomatis dapat 1 kode berbeda. PALING IDEAL.

KALAU TIDAK ADA fitur itu (paling umum) → pakai cara MANUAL (cocok utk batch awal):
  1. Simpan daftar 30 kode (bawah) di Google Sheet / catatan.
  2. Tiap ada order LUNAS (notif Scalev/WA), kirim 1 kode ke pembeli (template C).
  3. Coret/centang kode yang sudah dikirim biar tak dobel.
  (Volume awal kecil "27 pertama" → manual aman & cepat.)

CEK KODE TERPAKAI kapan saja: minta aku query DB (license_keys) — yang sudah
dipakai pembeli otomatis tercatat (uses bertambah). Stok menipis → minta aku
generate batch baru.
